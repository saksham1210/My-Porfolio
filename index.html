Plants Vs Zombies

Developed own clone of the Plants vs. Zombies game using Java (JDK-11) and JavaFX (2.0).

Bonus Functionality

We have made a special sun, which will drop in between the game randomly and will give you a super boost!.
The normal sun gives you +25 but this will make the value of Sun tokens equal to 200 no matter what the previous value was.

Musical Dance Floor

A euphoric user human-computer interaction experience which literally enables you to dance to your own sound. The room resonates with music produced according to the user movement on the floor.

The project is completely made using the Arduino and Basic Ultrasonic Sensors and some basic libraries to play the sounds.
